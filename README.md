# Chrome Extensions Manifest V2 数据分析

Chrome Web Store 上基于 Manifest V2 的扩展程序数据分析面板。

## 功能

- 84,446 款 Chrome V2 扩展的完整数据
- 按分类、安装量、更新时间、开发者类型、评分等多维度筛选
- 数据分析和洞察
- 机会发现：发现高安装低评分的替代机会
- 支持导出 CSV

## 数据来源

数据来源于 Chrome Web Store，包含所有 Manifest V2 扩展的信息。

## 使用

通过 GitHub Pages 访问，或直接在本地用 HTTP 服务器打开：

```bash
python -m http.server 8080
```

然后访问 http://localhost:8080
