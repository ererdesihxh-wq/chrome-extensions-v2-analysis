# Chrome Extensions Manifest V2 数据分析

Chrome Web Store 上基于 Manifest V2 的 84,446 款扩展程序数据分析面板。

## 功能

- **数据浏览** — 按安装量、评分、更新时间等排序，多条件筛选
- **分类筛选** — 功能分类、安装量范围、更新年份、开发者类型(公司/个人)、国家、评分范围
- **中文排除** — 可排除中文描述扩展
- **数据分析** — 功能分类分布、安装量排行、开发者类型统计、更新趋势
- **机会发现** — 高安装低评分替代机会、MV2 淘汰商机分析、市场评估
- **导出 CSV** — 筛选结果一键导出

## 本地启动

### 方法一：双击启动（推荐）
双击 `启动服务器.bat`

### 方法二：命令行
```bash
python -m http.server 8080
```
然后浏览器打开 http://localhost:8080

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 数据分析网页（主文件） |
| `extensions_data.json` | 84,446 条扩展数据 |
| `启动服务器.bat` | Windows 一键启动脚本 |
